🐧 Linux Ubuntu + VS Code 💻

Guía de Inicio Rápido Python 🐍

Configura tu entorno interactivo. ¿No tienes contraseña de administrador? No hay problema, sigue el Plan B.

Los 3 Pasos del Éxito

Paso 1 💻

Opción A: Terminal

Si tienes la contraseña del equipo, usa estos comandos para una instalación global:

# Ctrl+Alt+T

sudo apt update

sudo apt install python3 python3-ipykernel -y

⚠️

¿Sin contraseña? (Plan B)

¡Salta este paso! No escribas nada en la terminal. Ve directamente al Paso 2. VS Code instalará lo necesario en tu cuenta más adelante.

Paso 2 🔌

Extensiones de VS Code

Busca e instala estas dos extensiones oficiales de Microsoft (Ctrl+Mayús+X). Esto no pide contraseña.

🐍 Python (Microsoft)
📓 Jupyter (Microsoft)

Indispensable tanto para la Opción A como para el Plan B.

Paso 3 🚀

¡Ejecutar y Validar!

Abre tu archivo .ipynb y pulsa el botón Play.

▶

Si sigues el Plan B:

Aparecerá un mensaje:
"IPyKernel not installed".
Haz clic en el botón azul "Install".

✓ ¡Todo listo para programar!

💡 Consejo: El "Plan B" instala los archivos en tu carpeta personal. Tarda 1 minuto más la primera vez, pero funciona igual de bien que la instalación global.
